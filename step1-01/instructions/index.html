Configuration & Setup

Download and install:
  1. Node.js (includes npm):
    • https://nodejs.org/en/download/
    • Download the latest stable release of Node version 10 which includes npm version ^6.9.0
  2. .Net
    • https://www.python.org/downloads/
    • .Net comes preinstalled with Windows 10,
    • but you can find instructions on installing the required version here:
      • `
    • https://docs.microsoft.com/en-us/dotnet/core/install/windows?tabs=net60
  3. Visual Studio 2017 or Build Tools for Visual Studio 2017
    • https://my.visualstudio.com/Downloads
    • If using Build Tools, include the Visual C++ build tools workload and the optional Windows 10 SDK package.
    • If Visual Studio 2019 is installed run 
      npm config set msvs_version 2017
  4. Clone the codebase
    • https://experimentation.visualstudio.com/Analysis%20and%20Experimentation/_git/AnE.ExP.ConsumptionWebJobs?version=GBmaster
    • It is also possible to clone directly to Visual Studio using the following:
    • Clone to Visual Studio
  5. install 'vsts-npm-auth' and authenticate to ADO artifacts for npm packages:
    • npm install vsts-npm-auth -g --registry https://registry.npmjs.com
    • vsts-npm-auth -config .npmrc
  6. install yarn:
    • npm install yarn -g
    • Read more about Yarn specific commands.
    • https://dev.azure.com/experimentation/Analysis%20and%20Experimentation/_wiki/wikis/AnE.wiki/5529/Yarn
  7. install lerna:
    • npm install lerna -g
    • Read more about Library Development (Lerna) specific commands.
    • https://dev.azure.com/experimentation/Analysis%20and%20Experimentation/_wiki/wikis/AnE.wiki/5518/Library-Development

Open Workspace

In VSCode, select File > Open Workspace selecting the workspace file located in the ExperimentationUI repo /.vscode/ExperimentationUI.code-workspace
