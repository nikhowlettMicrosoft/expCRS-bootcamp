Configuration & Setup

Download and install:
  1. Node.js (includes npm):
    • https://nodejs.org/en/download/
    • Download the latest stable release of Node version 10 which includes npm version ^6.9.0
  2. .Net
    • https://www.python.org/downloads/
    • .Net comes preinstalled with Windows 10,
    • but you can find instructions on installing the required version here:
      • `
    • https://docs.microsoft.com/en-us/dotnet/core/install/windows?tabs=net60
  3. Visual Studio 2019
    • https://my.visualstudio.com/Downloads
    • If using Build Tools, include the Visual C++ build tools workload and the optional Windows 10 SDK package.
    • If Visual Studio 2019 is installed run 
      npm config set msvs_version 2017

Get App Configuration

The first step to getting the app configuration is getting JIT for the associated azure subscription with the App Service / Web Job. For Consumption that will be ConsumptionRestService AME Subscription.

Getting ready to setup

Procuruing the PPE Config
    • Navigate to the debug console for the ppe web jobs from Edge Browser: https://consumption-jobs-ame.scm.azurewebsites.net/DebugConsole
    • Navigate to: site\wwwroot\app_data\jobs[continuous or triggered][Webjob You are trying to test] And copy the [Webjob].exe.config file. Paste this file to your App.config file of the webjob you are trying to test.
