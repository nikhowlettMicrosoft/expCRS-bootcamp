Steps to send data to our Web Jobs
  1. First we will need a Visual Studio Solution to fire scorecard events. The repo is linked below.
        2. Make sure the client is connecting to the same topic as in your web job.
      • The setting for the Event solution is in Program.cs in the FireEventsForScorecardList function.
      • The location for the topic name for the Web Job may vary between web jobs, but usually can be found in App.config
        Make sure the client is connecting to the same service bus as in your web job.
      • The setting for the Event solution is in Program.cs in the Constants.cs
      • The location for the Service Bus connection for the Web Job can usually can be found in App.config
  2. Add a breakpoint in the *MessageHandler.cs file for the Web Job and run the specified Web Job.
    • With the Job running and breakpoint set,
    • Run the 'FireScorecardEvents' Solution.
    • If the breakpoint is hit, Congratulations, we have successfully received data!
